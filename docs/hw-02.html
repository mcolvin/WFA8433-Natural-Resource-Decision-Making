Exercise 1 Working with stochastic values and expectations

  1. Generate a vector named Y with 1001 stochastic replicates for a normal distribution with mean 10 and standard deviation of 1.
  2. Calculate the mean and standard deviation of Y and assign them as Y_mean and Y_sd respectively.
  3. Make a histogram of Y, labeling the x-axis as Y values and the y-axis as Frequency
  4. Calculate the probability that Y is less than 11. (HINT: you can use the sort() function to sort a vector in ascending order; ?sort)

Exercise 2 Making a function to do useful things

  1. Make a function that multiplies an input vector by 2, sums the vector and then returns that sum.
  2. Run the function created in 2a on the vector Y created in 1a.
  3. Create another function called risk that returns the probability of an achieving an expected outcome or less from a vector of stochastic replicates. (HINT: subset() and length() functions might be useful).
  4. Run the function created in 2d on the vector Y created in 1a for an outcome of 11. How does it compare to the value you calculated for 1d?

Exercise 3 Linear modeling and prediction

  1. Read in the dataset elevation-density.csv and assign itele_data.
  2. Plot density (y-axis) versus elevation (x-axis) and label each axis as you see fit.
  3. Create the same plot as above, but make group 1 red, group 2 blue, and group 3 green.
  4. Fit a linear model that predicts density from elevation, group, and an interaction of group and elevation.
  5. Plot the residuals (y-axis) versus the predicted to evaluate whether you have violated any assumptions and a plot of the observed density versus the fitted values. (HINT: recall resid() and fitted())
  6. Now create another field in ele_data named ldensity that is the natural log of density.
  7. Refit the linear model from 3a using ldensity.
  8. Plot the residuals (y-axis) versus the predicted (x-axis) to evaluate whether you have violated any assumptions and a plot of ldensity versus the fitted values.
  9. Create a dataset named predicted_data using the expand.grid() function to predict values for the range of elevations in ele_data in increments of 1 and each group.
  10. Add columns to predicted_data for the predicted value, lower 95% prediction interval and upper 95% prediction interval.
  11. Make a plot of predicted values and 95% prediction intervals for group 2 as lines. The line for the predicted value should be solid and the lines for the prediction intervals should by dotted. (Recall that lty=1 is a solid line and lty=2 is a dotted line)
